# Roadmap

Vizzor is in early alpha. The goal is to move from a working local MVP to a reliable self-hosted BI portal.

## Alpha

- FastAPI backend shell
- React portal shell
- dashboard catalog API
- Tableau connector placeholder
- Docker Compose local environment
- basic JWT demo flow

## MVP

- persistent dashboard catalog
- role-based dashboard visibility
- real Tableau Connected Apps token signing
- production-ready auth strategy selection
- embed error states and audit-friendly logs
- deployment guide

## Later

- Power BI connector
- Metabase connector
- Superset connector
- theme token editor
- managed hosted version exploration
