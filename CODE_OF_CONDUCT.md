# Code of Conduct

Vizzor is a community project for people building open, self-hostable analytics infrastructure.

We expect contributors to be respectful, constructive, and considerate. Disagreement is welcome when it improves the work; harassment, insults, discrimination, and personal attacks are not.

If you experience behavior that makes the project unsafe or unwelcoming, please open a private maintainer contact channel when one is published. Until then, avoid sharing sensitive personal information in public issues.

Maintainers may remove comments, close threads, or restrict participation when behavior harms the project community.
